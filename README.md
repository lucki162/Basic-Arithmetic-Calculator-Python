# Simple Calculator

This is my first Python program! It's a simple command-line calculator 
that performs addition, subtraction, multiplication, and division.

## How to Run
1. Make sure Python is installed
2. Run the script: `python calculator.py`
3. Enter two numbers when prompted
4. The program will show the result of +, -, *, /
5. Type "Y" to stop or "N" to continue
